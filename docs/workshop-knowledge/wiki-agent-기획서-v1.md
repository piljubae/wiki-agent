## 배경 및 목적

사내 Confluence 위키는 방대하지만 검색 품질이 낮아 원하는 문서를 찾는 데 평균 8~12분이 소요됨.
팀원들이 Slack에서 "이거 어디 있어?"를 반복하는 패턴을 관찰 → 자동화 가능성 확인.

**목적:** Slack에서 자연어로 질문하면 Confluence 문서를 찾아 답변해주는 봇 개발.

## 범위 (v1.0)

- Slack Socket Mode 연동
- Confluence CQL 검색
- Claude Haiku 기반 답변 생성
- 단일 스페이스(ProductApp) 지원

## 제외 범위 (v1.0)

- RAG/벡터 검색 → v2에서
- GitHub Wiki 검색 → v2에서
- 대화 히스토리 → v2에서
- 다중 스페이스 → v2에서

## 성공 지표

| 지표 | 목표 |
|------|------|
| 검색 소요 시간 | 8분 → 30초 이하 |
| 정확도 (Recall@5) | 70% 이상 |
| 일평균 질문 수 | 10건 이상 |

## 기술 스택

- **언어:** Kotlin (JVM)
- **프레임워크:** Koog 0.7.x
- **LLM:** Anthropic Claude Haiku
- **검색:** Confluence REST API (CQL)
- **배포:** Docker + 사내 서버
