- ✅ Confluence CQL 검색 (다중 스페이스)

## 5월 (v0.2.x)

### 대화 히스토리
- in-memory 3턴 유지
- Sliding window 압축 (토큰 절약)
- "아까 말한 거" 같은 후속 질문 처리

### RAG (ChromaDB)
- Confluence 문서 벡터 인덱싱
- 의미 기반 검색 fallback
- 임베딩: Google Embedding API 또는 LLM Expand

### 프로젝트 메모리
- `.wiki/project-memory.md` 상시 주입
- 팀/도메인 컨텍스트 봇에 고정

## 6월 (v0.3.x)

### A2A 멀티 에이전트
- ConfluenceAgent ← OrchestratorAgent
- Specialist 에이전트 분리 (검색/요약/번역)
- A2A Protocol 연동

### 자동 인덱싱
- Confluence 변경 감지 → 자동 재인덱싱
- 스케줄러 (매일 새벽 2시)

### Reaction 피드백
- 👍 / 👎 이모지 → 검색 품질 피드백
- 낮은 평점 케이스 → 골든 데이터셋 자동 추가
